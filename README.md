# simplestock
A simple macOS app for monitoring specified stock prices on the desktop


### 使用说明

右键单击stock.app, 选择 显示包内容， 打开 Contents 文件夹，用任意文本编辑器打开 stocklist.txt, 股票代码每行一个，可在新浪财经股票页面查看股票代码。

Right-click stock.app, select Show Package Contents, open Contents folder, open stocklist.txt with any text editor, stock codes are one per line, you can view stock codes in Sina Finance stock page.
